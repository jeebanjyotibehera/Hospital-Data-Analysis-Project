# Hospital Data Analysis Project

## Project Description
This project focuses on analyzing hospital patient data to understand
patient conditions, treatment costs, and overall trends.

## Tools Used
- Python (Pandas, Matplotlib)
- SQL
- Excel
- Power BI

## Project Structure
- data : cleaned hospital dataset
- python : data analysis and visualization code
- sql : SQL queries for analysis
- powerbi : Power BI dashboard file
- visuals : graphs generated using Python

## Key Insights
- Average treatment cost varies by medical condition
- Patient age distribution shows majority of patients are adults
- Certain conditions have higher patient counts

## Dashboard
An interactive Power BI dashboard is included for better visualization.
